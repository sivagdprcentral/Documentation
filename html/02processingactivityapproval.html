How do I request for approval or approve a Processing Activity?Â¶

As Processing Activity is an important record mandated by GDPR regulations, it is important that any Processing Activity that is created or changed also goes through a formal approval process.

Note

If a change is made to a previously approved activity it will lose its approved status and must be reverified and reapproved before it can be considered the latest approved version.

You can approve a Processing Activity in GDPR Central by performing the following steps:

  1. Login to GDPR Central using your credentials if not already logged in.

  2. In the left pane, under your organization, click Processing Activities.

  3. A list of all Processing Activities for the customer are listed along with their statuses.

  4. Click the View button () for the Processing Activity that needs to be approved.

  5. Processing Activities Details page appears.

  6. If you would like to see what has changed since the previous version of the Processing Activity, click Compare button.

  7. If you have logged in to the system with administrative privileges, you will be able to approve the Processing Activity. Click Approve Processing Activity button.

    If you have logged in to the system with user privileges, you will be able to request for approving the Processing Activity. Click Ask for Approval button.

  8. If you have approved the Processing Activity, you will get a success message if your approval is successful.

    If you have requested for approval for the Processing Activity, you will get a success message if your request for approval has been successfully sent.
